# DigiTools - Digital Product Platform

## Project Description
DigiTools is a modern web application where users can explore and purchase digital products like AI tools, templates, and assets. It includes a dynamic cart system and responsive UI design.


### Technologies Used
1. React.js
2. Tailwind CSS
3. DaisyUI
4. JavaScript (ES6+)
5. React-Toastify

#### Features

1. Add to Cart System
Users can add products to the cart and see total count in the navbar

2. Product & Cart Toggle
Users can switch between product view and cart view using toggle buttons

 3. Cart Total & Checkout
Cart shows total price and allows removing items or clearing all via checkout.
